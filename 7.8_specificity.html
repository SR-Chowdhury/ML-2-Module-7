Specificity is the algorithm used by browsers to determine the CSS declaration that is the most relevant to an element, which in turn, determines the property value to apply to the element.

  • First, it gives priority external css
  • Second, it will check is there any internal CSS; if then it will replace first one
  • Third, it will check is there any embedded or inline CSS; if then it will be the final
  • Forth, it will check if there any css properties value is used as !important; then it will be the final and rest of all are ignored.
